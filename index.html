Semantic HTML5 Elements

Why proper structure matters for modern websites

Key Semantic Tags

HTML5 introduced several new semantic elements that help define the structure of web pages:

  • <header> - Introductory content
  • <nav> - Navigation links
  • <main> - Primary content
  • <article> - Self-contained composition
  • <section> - Thematic grouping
  • <footer> - Closing content

Web Accessibility

Making the web usable for everyone

Accessibility Features

Proper HTML structure is the foundation of web accessibility:

  • Use landmark roles (implied by semantic elements)
  • Provide text alternatives for images
  • Ensure proper heading hierarchy
  • Use ARIA attributes when necessary

SEO Best Practices

Helping search engines understand your content

On-Page SEO Techniques

Search engines rely on HTML structure to interpret content:

  • Use descriptive, keyword-rich titles
  • Maintain proper heading hierarchy (h1-h6)
  • Include meta descriptions
  • Use semantic markup for content
  • Provide alt text for images
